RankSVM-PTM

RankSVM-PTM: is a tool for predicting acetylation and succinylation in proteins. The program is implemented in Matlab.


Reference

Yan Xu, Zu Wang, Yingxi Yang, Yuanhai Shao. Prediction of acetylation and succinylation in proteins based on multi-label learning RankSVM [J]. Submitted,2017

Yan Xu, Yingxi Yang, Xingyan Li. Prediction of Human Serine, Threonine and Tyrosine Phosphorylated Proteins Based on Multi-label Learning[J]. Submitted,2018



  • 1. Operating environment:

    The operating system of installed MATLAB.

    • 2. File description:

      RankSVM-PTM.zip is main program.

      • 3. Instructions:

        Just run the example1.m program (AAC.mat,AAindex.mat,CKSAAP.mat are three feature extraction results about benchmark dataset in this paper)

        • 4. Software and Data

          Download from:[RankSVM-PTM.zip]
