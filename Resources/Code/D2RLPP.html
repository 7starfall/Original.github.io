2DRLPP

A Demo Matlab code for 2DRLPP for noisy image recognition. (You could Right-Click [Code] , and Save, then you can download the whole matlab code.)

Reference

Wei-Jie Chen, Chun-Na Li, Yuan-Hai Shao, Ya-Xin Shi and Nai-Yang Deng. 2DRLPP: Robust Two-Dimensional Locality Preserving Projection with Regularization. Submitted. 2018.


The code is tested by Matlab 2018a. Any question or advice please email to wjcper2008@126.com.
